## What Scout Does

Scout is an autonomous AI agent that handles the entire real estate buyer outreach workflow:

1. **Analyzes** buyer preferences using LLM
2. **Matches** properties with scoring algorithm
3. **Generates** personalized HTML emails
4. **Sends** via SendGrid automatically

One click. Multiple buyers. Zero manual work.
